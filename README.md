# Filtering HTTP proxy

## General
Proxy is written in [Lua language](http://www.Lua.org)

It allows customize the filtering of requests passing via proxy

## Running

### Linux/OSX:
lua run.lua

### Windows:
murgalua.exe proxy.lua
